KWP SBB for Woocommerce plugin helps you to go back to category product page store from detail page of each item in woocommerce.

== Description ==

In woocommerce, Category product page could consists of several pages, e.g 121 pages, when the customer goes to
page 93 for instance, an item is selected and details page of that item is being viewed, then the customer 
wants to go back to the same page (this means page 93) instead of page 1 again. This plugin adds a store back button
to product details page of woocommerce plugin!

= Plugin Features =

* You may specify font-size, font-family, text-color and background color of the button to be added to product's detail page in wc.


== Installation ==

1. Upload `kwpsbb-for-woocommerce.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
